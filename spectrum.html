INTRODUCTION

Using Arduino and ultrasonic sensors, an interactive colour making installation was created that uses human interaction to create unique colours, and teaches our audience about the RGB colour space. 3 Ultrasonic sensors each control an RGB Value of the led lights, users can control the colour hues according to the distance that they are above the sensor. The higher they hover over a sensor, the brighter the colour. This project engages users as well as teaches them about colour mixing, all while emitting a beautiful glow and calming effect.

PROBLEM

Our final concept of the Interactive RGB Generator was born through iterating on our previous concepts. We fell in love with the impact that colour had and wanted to create an engaging experience that pushes the concept of colour further and also helps to solve a problem.

OUR GOAL

We created Spectrum as a tool that allows users to mix different color values of RGB by hovering over three ultrasonic sensors, each one controlling one of these values.

By hovering over multiple ultrasonic sensors users are able to learn about mixing colours. It is a fun and engaging, and beautiful installation that ultimately can be used as an education tool.

[ PROCESS ]

fabrication

For the final iteration we laser cut a wooden frame for the light box. We created a box that has negative space on the edges for visual interest, and hid the led light strip under the wooden frame. We also created ultrasonic sensor cases, and laser cut the frame of our box. 

[ PROCESS ]

user experience

The way that the project was assembled and presented, strongly affects the user experience. In order to get the strongest glowing effect we created "paddles" that the user holds over the sensors. As we were testing the sensors with our hands, it was finicky because of the gaps between our fingers, using a flat and solid surface was an easy way to fix this problem. A possible caveat could be that there are three sensors, and a user has only two hands so they are not able to hover over all three sensors at once, which limits their colour combination. All though this could be seen as a negative, it can also be seen as a positive, as it encourages social interaction and teamwork with another user.

[ PROCESS ]

arduino code

We mapped the distance that the ultrasonic sensor measures to the all of the 255 values in the RGB spectrum. The mapped distance only covers 30 cm  because this is the range that fits best with the users range of motion.

Further, we included all the different combinations that the user could do using a various If-Statements.

[ PROCESS ]

further development

If we were to continue this project we would love to see it on a larger scale, covering the surface of a wall. It would also be a great learning tool for children if we could display the RGB values on a lcd screen, which would consist of simply outputting the color data from the serial port.
