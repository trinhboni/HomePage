SPECTRUM

[ 2017 ]

GOALS

We created Spectrum as a tool that allows users to mix different color values of RGB by hovering over three ultrasonic sensors, each one controlling one of these values.

It is a fun and engaging, and beautiful installation that ultimately can be used as an education tool.
