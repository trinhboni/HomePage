manga rock

INTRODUCTION

Manga Rock is an online manga reader that is available for both IOS and Android. There are about 2 millions users on IOS before it hit Android. The App supports more than 7 languages ( English, Italian, Vietnamese, German, French, Spanish and Chinese ) and there are serveral manga sources for each languages with over 100,000 titles. Users are able to download manga and use it on the go.

UX PROBLEM

Many users has complained about the screen size not being compatible on some devices. Sometimes pictures become to zoomed out/in which makes it harder to read. 

UI PROBLEM

Three UI problems that exist within the app include: 

“ I want to know the action is done ”

“ I want the interaction to feel natural

“ I don’t want to make my screen a mess ”

THE GOAL

The main goal for the project is to enhance the user experience by improving the app's UI motion design. From my research, the app interface was very messy and all over places, it also does not have a loader. I decided to create two short animations that could solve these UI problems.
