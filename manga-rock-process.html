manga rock

INTRODUCTION

Manga Rock is an online manga reader that is available for both IOS and Android. There are roughly around 2 millions users on IOS before it hit Android.

Features that makes Manga Rock standouts:

The App supports more than 7 languages ( English, Italian, Vietnamese, German, French, Spanish and Chinese ).

There are serveral manga sources for each languages with over 100,000 titles.

Download feature allows user read on the go.


UX PROBLEM

Many users has complained about the screen size is not compatible on some devices. Sometimes pictures was too zoomed out/in which makes it harder to read. Also, there are some glitches and while using the app features.

UI PROBLEM

3 main UI problems:

“ I want to know the action is done ”

“ I want the interaction to feel natural

“ I don’t want to make my screen a mess ”

THE GOAL

The main goal for the project is to enhance the user experience by improving the app's UI motions. From my research, the app interface was very messy and all over places and it does not have a loader. Therefore, I decided to create short animations that could solve these problem.

PROCESS

[ PROCESS ]

primary and secondary research

Before going straight in developing the motions, I did some researched about the app first.

[ PROCESS ]

initial sketches

My notes and sketches while developing and recreate the app's interface.
[ PROCESS ]

inspirations

In order to animate the scenery grow up naturally and nicely, I needed to split each object into multiple layers and worked with each layer in After Effect. Throughout the process, I used different easing expressions, plugins and turorials to complete the project.

[ PROCESS ]

story boards ( sketches )

After doing some researches and coming up with ideas, I move to drafting out some story board for the animation.

[ PROCESS ]

story board / mockups

After sketched all of the element that I needed. I moved all the sketches to Illustrator and start tracing/ making all of them in vector form so I can animated it in After Effect. 

FINAL ANIMATION

REFLECTION

After this project, overall I found it was very enjoyable with learning new techniques in Adobe After Effect and Character Animator CC.

One thing that I found challenging for me was when I worked on the character itself. Since I it was my first time using Adobe Character Animator, there were a learning curve throughout the process. However, with online tutorials and extensive iterations, I was able to grasp the basic knowledge of the program. which allowed me to adventure more into its features.

Overall. this project has given me the opportunity to expand my skills in motion graphic design which can be useful in future project.
